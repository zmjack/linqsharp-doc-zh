## 自动字段

| 图例               | 描述     |
| ------------------ | -------- |
| :pencil2:          | 自动管理 |
| :heavy_check_mark: | 允许生效 |
| :x:                | 禁止生效 |
| :warning:          | 抛出错误 |

<br/>

### 自动特性

**AutoCreationTime** / **AutoLastWriteTime**

|               | 新增               | 修改               | 删除               |
| ------------- | ------------------ | ------------------ | ------------------ |
| **Auto**      | :pencil2:          | :pencil2:          | :heavy_check_mark: |
| **Preserve**  | :heavy_check_mark: | :x:                | :heavy_check_mark: |
| **Specified** | :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: |

<br/>

**RowLock**

**数据库值** 为 **null** 时：

|               | 新增               | 修改               | 删除               |
| ------------- | ------------------ | ------------------ | ------------------ |
| **Auto**      | :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: |
| **Preserve**  | :heavy_check_mark: | :x:                | :heavy_check_mark: |
| **Specified** | :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: |

**数据库值** 为 **not null** 时：

|               | 新增               | 修改               | 删除               |
| ------------- | ------------------ | ------------------ | ------------------ |
| **Auto**      | :heavy_check_mark: | :warning:          | :warning:          |
| **Preserve**  | :heavy_check_mark: | :warning:          | :heavy_check_mark: |
| **Specified** | :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: |
