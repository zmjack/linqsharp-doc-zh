## 自动字段

| 图例 | 描述     |
| ---- | -------- |
| ✏️    | 自动管理 |
| ✔️    | 允许生效 |
| ❌    | 禁止生效 |
| ⚠️    | 抛出错误 |

<br/>

### 自动特性

#### 创建时间 / 最近修改时间

**AutoCreationTime** / **AutoLastWriteTime**

|              | 新增 | 删除 | 修改 |
| ------------ | :--: | :--: | :--: |
| **Auto**     |  ✏️   |  ✔️   |  ✏️   |
| **Preserve** |  ✏️   |  ✔️   |  ❌   |
| **Free**     |  ✔️   |  ✔️   |  ✔️   |

<br/>

#### 锁定字段

**RowLock**

- 当字段值为 **not null** 时，禁止指定列的值发生变化。

**数据库值** 为 **null** 时：

|              | 新增 | 删除 | 修改 | 修改指定列 |
| ------------ | :--: | :--: | :--: | :--------: |
| **Auto**     |  ✔️   |  ✔️   |  ✔️   |     ✔️      |
| **Preserve** |  ✔️   |  ✔️   |  ✔️   |     ✔️      |
| **Free**     |  ✔️   |  ✔️   |  ✔️   |     ✔️      |

**数据库值** 为 **not null** 时：

|              | 新增 | 删除 | 修改 | 修改指定列 |
| ------------ | :--: | :--: | :--: | :--------: |
| **Auto**     |  ✔️   |  ⚠️   |  ⚠️   |     ⚠️      |
| **Preserve** |  ✔️   |  ⚠️   |  ❌   |     ❌      |
| **Free**     |  ✔️   |  ✔️   |  ✔️   |     ✔️      |
