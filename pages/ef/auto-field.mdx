## 自动字段

| 图例 | 描述     |
| ---- | -------- |
| ✏️    | 自动管理 |
| ✔️    | 允许生效 |
| ❌    | 禁止生效 |
| ⚠️    | 抛出错误 |

<br/>

### 自动特性

**AutoCreationTime** / **AutoLastWriteTime**

|               | 新增 | 修改 | 删除 |
| ------------- | ---- | ---- | ---- |
| **Auto**      | ✏️    | ✏️    | ✔️    |
| **Preserve**  | ✔️    | ❌    | ✔️    |
| **Specified** | ✔️    | ✔️    | ✔️    |

<br/>

**RowLock**

**数据库值** 为 **null** 时：

|               | 新增 | 修改 | 删除 |
| ------------- | ---- | ---- | ---- |
| **Auto**      | ✔️    | ✔️    | ✔️    |
| **Preserve**  | ✔️    | ❌    | ✔️    |
| **Specified** | ✔️    | ✔️    | ✔️    |

**数据库值** 为 **not null** 时：

|               | 新增 | 修改 | 删除 |
| ------------- | ---- | ---- | ---- |
| **Auto**      | ✔️    | ⚠️    | ⚠️    |
| **Preserve**  | ✔️    | ⚠️    | ✔️    |
| **Specified** | ✔️    | ✔️    | ✔️    |
